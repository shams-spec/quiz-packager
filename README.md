# Quiz Packager

Quiz Packager combines two `.docx` quiz files into one downloadable Word document.

It runs in the browser. Files are processed locally in the visitor's browser and are not stored in a database.

## Files

- `index.html` - the app
- `jszip.min.js` - local helper used to read and create `.docx` files

## Deploy

This folder can be deployed as a static site on Vercel.
